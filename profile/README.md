### 风险一：IP 被封锁

这是国内用户使用境外 VPS 最常遇到的问题。一旦 IP 被封，轻则访问变慢，重则完全失联。

**应对方案：**
- 选择支持免费换 IP 的套餐（SLA 系列每两周一次免费更换）

共享宿主机的 VPS，如果遇到"坏邻居"大量占用资源，会明显影响自己的使用体验。

**应对方案：**
- SLA 系列提供**独享 CPU 资源**，从架构层面避免邻居效应
- CN2 GIA-E 系列虽然是共享型，但搬瓦工的超售率管控相对克制，用户口碑普遍良好

### 风险三：限量版套餐断货导致无法续费

限量版套餐因为性价比极高，偶尔会出现"卖一台少一台"的局面。不过搬瓦工的政策是：**已购套餐可以正常续费**，并不会因为库存耗尽而强制停服。

### 风险四：错过补货窗口，被迫以高价购买常规套餐

这是限量版用户最痛的点。补货窗口短则几小时，长也不超过两三天。
